# gulp3-project

> 一个基于gulp3构建的项目

## 构建步骤

### 全局安装gulp
```
(sudo) npm i -g gulp@3
```

### 安装项目依赖
```
npm install
```

### 开发环境
```
npm run dev
```

### 生产环境
```
npm run build
```
